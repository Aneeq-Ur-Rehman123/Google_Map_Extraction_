# Google_Map_Extraction_
Business Data Scraper and AI Description Generator
This project automates the process of collecting business data based on specified parameters such as business type, country, and ZIP codes. It efficiently fetches business listings across multiple pages, ensures unique entries by tracking IDs, and generates AI-powered descriptions for each business. The collected data is then compiled into a CSV file for further use or analysis.

Features

Configurable input parameters for business type, country, ZIP codes, and pagination

Deduplication of business records using unique identifiers

AI-driven description generation for enriched business profiles

CSV export of collected and processed data for easy integration

(Optional) Integration with Google Sheets for direct data upload

Output directory management and cleanup utilities

Ideal for data analysts, marketers, or developers needing structured business information with enhanced descriptions for applications such as directories, marketing campaigns, or research.
